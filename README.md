### disease_prediction
A Machine Learning-based web application that predicts the likelihood of diabetes using user health data. Built using Logistic Regression, Decision Tree, and Streamlit for an interactive interface.
This project is a Machine Learning-based web application that predicts the likelihood of diabetes based on user input features such as glucose level, BMI, age, and other medical parameters.
The system uses classification algorithms like Logistic Regression and Decision Tree to analyze the data and provide predictions. The model is trained on the PIMA Indians Diabetes Dataset, which contains real-world medical data.

### Features
1.Predicts diabetes risk based on user input.
2.Interactive web interface using Streamlit.
3.Uses multiple ML algorithms for comparison.
4.Displays accuracy and confusion matrix.
5.Easy to use and beginner-friendly.

### Project Objective
The main goal of this project is to demonstrate how machine learning can be used in healthcare for early disease prediction and decision-making.
This project is a Machine Learning-based web application that predicts the likelihood of diabetes based on user input features such as glucose level, BMI, age, and other medical parameters.

### How It Works
1. User enters health-related data (e.g., glucose level, BMI, age)
2. Data is processed and passed to  ML model
3. Model predicts whether the person is likely to have diabetes
4. Result is displayed instantly on the web app.
